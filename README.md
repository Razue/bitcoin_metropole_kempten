## Was ist die Bitcoin Metropole?

Die Bitcoin Metropole Kempten ist ein offener Treffpunkt für Wissensaustausch, Bildung und kulturelle Projekte rund um Bitcoin. Betrieben vom Allgäuer Bitcoin-Künstler **MX12 ART** und unterstützt von der lokalen Bitcoin-Community.
